# 图标文件说明

## 需要的图标文件

请在此目录下添加以下PNG格式的图标文件：

- `icon16.png` - 16x16像素，用于扩展管理页面
- `icon48.png` - 48x48像素，用于扩展管理页面
- `icon128.png` - 128x128像素，用于Chrome应用商店

## 图标设计建议

- 使用🔐锁或🔑钥匙相关的图标
- 主色调建议使用蓝色或紫色系
- 背景建议使用透明或简洁的纯色
- 确保图标在小尺寸下仍清晰可见

## 临时替代方案

如果暂时没有图标文件，可以：
1. 从manifest.json中临时移除icons相关配置
2. 或使用在线图标生成器创建简单图标

## 推荐的图标生成工具

- [Favicon Generator](https://favicon.io/)
- [Icon Generator](https://icon-generator.net/)
- Canva 或 Figma

## 图标示例概念

```
🔐 CryptoToolkit
- 主图标：盾牌 + 锁
- 颜色：渐变蓝紫色 (#4f46e5 到 #7c3aed)
- 风格：现代简约
```
